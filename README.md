# Proba IT 2022

## Pentru rularea frontend-ului

    1) cd frontend
    2) npm i
    3) npm start
    4) serverul va fi rulat pe localhost:3000
    5) deschideti http://localhost:3000 pentru a vedea in browser

## Pentru rularea backend-ului

    1) cd backend
    2) npm i
    3) npm start
    4) serverul va fi rulat pe localhost:3000
    5) deschideti http://localhost:3000 pentru a vedea in browser
